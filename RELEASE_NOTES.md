# Raw Accel Reimagined 1.8.2

## English

This release focuses on making curve editing reliable and the application easier to understand.

- Fixed LUT decimal parsing on Windows installations that use a comma as the decimal separator.
- Fixed LUT points disappearing, changing, or returning to the bottom after **Save & Apply**.
- Preserved the selected LUT point and curve preview after applying.
- Highlighted the final LUT point so users can clearly see where the curve ends.
- Added draggable Classic curve controls and corrected the Classic Shape and Classic Cap drag directions.
- Added a profile assignment notice showing which mouse is actually controlled by the selected profile.
- Changed Reset into **Restore Defaults**, with a safe preview that does not touch the file or driver until **Save & Apply**.
- Replaced Windows notification boxes and sounds with silent, themed in-app dialogs.
- Added a bilingual **What's New** pop-up after successful automatic updates.

User profiles, settings, themes, language preferences, and mouse associations remain preserved during automatic updates.

## Português

Esta versão melhora a confiabilidade da edição de curvas e deixa o funcionamento do aplicativo mais fácil de entender.

- Corrigida a leitura de decimais LUT no Windows quando a vírgula é utilizada como separador decimal.
- Corrigidos os pontos LUT que desapareciam, mudavam ou voltavam para baixo após **Salvar e aplicar**.
- Preservados o ponto LUT selecionado e a prévia da curva depois da aplicação.
- Destacado o ponto final LUT para deixar claro onde a curva termina.
- Adicionados controles arrastáveis para a curva Clássica e corrigidas as direções de Forma Clássica e Limite Clássico.
- Adicionado um aviso de associação mostrando qual mouse realmente é controlado pelo perfil selecionado.
- Redefinir foi substituído por **Restaurar padrões**, com uma prévia segura que não altera o arquivo nem o driver antes de **Salvar e aplicar**.
- As caixas e os sons de notificação do Windows foram substituídos por diálogos silenciosos e tematizados dentro do aplicativo.
- Adicionado um pop-up bilíngue de **Novidades** depois de atualizações automáticas concluídas.

Perfis, configurações, temas, idioma e associações de mouse continuam preservados durante atualizações automáticas.
