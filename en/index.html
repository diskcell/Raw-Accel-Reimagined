Installation and first steps

The project is still under development. Use only files from this repository and keep a backup of your configuration.

  1. Download or clone the official Raw Accel Reimagined repository.
  2. Install the driver with installer.exe and restart Windows when the installer requests it.
  3. Open RawAccelReimagined.exe, select a profile and use Save & Apply.
Raw Accel Reimagined preserves the original Raw Accel engine. This project focuses its changes on the interface, visualization, profiles and configuration tools.

Transparency and safety

Raw Accel uses a driver to transform mouse input on Windows. Download it only from the official repository, review the files before installing and avoid third-party repackaged downloads.

The close button minimizes the application to the system tray. To exit and disable acceleration, right-click the tray icon and select the close option.

Frequently asked questions

Quick answers about this project and its relationship with the original Raw Accel.

What is Raw Accel Reimagined?

It is an independent modern interface for configuring the Raw Accel driver, with an interactive graph, profiles, per-device settings, themes, and English and Portuguese support.

Does the project change the original formulas?

The calculations and primary driver communication continue to use the original engine. New features validate configurations through the Raw Accel wrapper before applying them.

Is this the official Raw Accel project?

No. Raw Accel Reimagined is an independent project based on Raw Accel and preserves the license and credits of the original work.

Where can I find the source code and complete instructions?

Visit the project repository on GitHub. You can also consult the original Raw Accel project.
