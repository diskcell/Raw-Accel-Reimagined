Instalação e primeiros passos

O projeto ainda está em desenvolvimento. Use somente os arquivos deste repositório e mantenha um backup das suas configurações.

  1. Baixe ou clone o repositório oficial Raw Accel Reimagined.
  2. Instale o driver com installer.exe e reinicie o Windows quando a instalação solicitar.
  3. Abra RawAccelReimagined.exe, escolha um perfil e use Salvar e aplicar.
O Raw Accel Reimagined preserva o motor original do Raw Accel. As mudanças deste projeto estão concentradas na interface, visualização, perfis e ferramentas de configuração.

Transparência e segurança

Raw Accel utiliza um driver para transformar a entrada do mouse no Windows. Baixe somente pelo repositório oficial, confira os arquivos antes de instalar e evite pacotes republicados por terceiros.

O botão de fechar minimiza o aplicativo para a bandeja. Para encerrar e desativar a aceleração, use o botão direito no ícone da bandeja e escolha a opção de fechar.

Perguntas frequentes

Respostas rápidas sobre o projeto e sua relação com o Raw Accel original.

O que é o Raw Accel Reimagined?

É uma interface independente e moderna para configurar o driver Raw Accel, com gráfico interativo, perfis, ajustes por dispositivo, temas e suporte a português e inglês.

O projeto altera as fórmulas originais?

Os cálculos e a comunicação principal com o driver continuam usando o motor original. Recursos novos validam as configurações pelo wrapper do Raw Accel antes de aplicá-las.

Este é o projeto oficial do Raw Accel?

Não. Raw Accel Reimagined é um projeto independente baseado no código do Raw Accel e mantém a licença e os créditos do trabalho original.

Onde encontro o código e as instruções completas?

No repositório do projeto no GitHub. Consulte também o Raw Accel original.
