RAW ACCEL REIMAGINED

ENGLISH
This folder contains the ready-to-run modern application and all runtime files it needs.
Run installer.exe once if the driver is not installed, restart Windows, and then open RawAccelReimagined.exe.
Personal settings.json, language, theme, profiles, and device associations remain inside this folder and are excluded from Git.
The legacy rawaccel.exe located at the repository root is not part of the modern application or its Release packages.

PORTUGUES
Esta pasta contem o aplicativo moderno pronto para uso e todos os arquivos necessarios durante a execucao.
Execute installer.exe uma vez se o driver ainda nao estiver instalado, reinicie o Windows e depois abra RawAccelReimagined.exe.
O settings.json pessoal, idioma, tema, perfis e associacoes de dispositivos permanecem nesta pasta e nao sao enviados ao Git.
O rawaccel.exe antigo localizado na raiz do repositorio nao faz parte do aplicativo moderno nem dos seus pacotes de Release.
